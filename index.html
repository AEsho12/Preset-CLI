How it works

Get started in under 2 minutes

Three simple commands to revolutionize your development workflow

1

Install globally

Install via npm and start using immediately. No setup required.

npm install -g preset-cli
2

Save your first preset

Navigate to any project directory and save it as a reusable template.

preset save my-template
3

Deploy anywhere

Load your presets into any directory to start new projects instantly.

preset load my-template
